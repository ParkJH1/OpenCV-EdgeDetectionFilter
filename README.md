# OpenCV-EdgeDetectionFilter
opencv를 이용한 에지 검출 필터 구현



## Python 환경

- anaconda 4.6.11
  - python 3.6
  - numpy 1.17.4
  - matplotlib 3.1.2
  - opencv-python 4.1.2.30



## 환경 설정

- conda 환경 생성 및 활성화

```
>>> conda create -n opencv-env python=3.6
>>> conda activate opencv-env
```



- python 모듈 설치

```
>>> pip install -r requirements.txt
```
